## Numpy <a name="numpy"></a>

1D array

```Python
a = np.array([1, 2, 3])
print(a)

######################
[1 2 3]
```

2D array

```Python
b = np.array([[1, 2], [3, 4]])

#######

[[1, 2]
 [3, 4]]
 
 ```

3D array

```Python
# Creating a 3D array from a list of lists of lists
c = np.array([[[1, 2], [3, 4]],
              [[5, 6], [7, 8]]])
print(c)

"""
[[[1 2]
[3 4]]
[[5 6] 
[7 8]]]
"""
```
